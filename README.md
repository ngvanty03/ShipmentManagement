# Project: ShipmentManagement

## 1. Overview
The web application which have two types of users: admin staff and customers (multi tenant). 
Core features include the ability for admins to create, update, and delete shipments
Customers can view and trace the status of their shipments. 
The app should be user-friendly and efficient, enhancing our logistics operations


## 2. Tech Stack

### Frontend
Use ReactJS/Typescript with tailwind css


### Backend
use .NET 9 API, EntityFramework, SQL server and follow CQRS pattern
